Schedula Backend API

A backend application for doctor and patient management built with Node.js, Express, TypeScript, Prisma, PostgreSQL, and JWT Authentication.

🚀 Live Deployment

https://schedula-priya.onrender.com

📌 API Endpoints
Authentication
Signup
POST /api/auth/signup

Login
POST /api/auth/login

Doctor Profile
Create Doctor Profile
POST /api/doctor/profile

Get Doctor Profile
GET /api/doctor/profile

Update Doctor Profile
PATCH /api/doctor/profile

Patient Profile
Create Patient Profile
POST /api/patient/profile
Get Patient Profile
GET /api/patient/profile
Update Patient Profile
PATCH /api/patient/profile
Doctor Discovery
Get All Doctors
GET /api/doctor
Search Doctors
GET /api/doctor?search=priya
Filter By Specialization
GET /api/doctor?specialization=cardiologist


⚙️ Local Setup
Clone Repository
git clone https://github.com/Priya-Kadamstudenthhyu/schedula-priya.git
Install Dependencies
npm install
Run Prisma Migration
npx prisma migrate dev
Start Development Server
npm run dev

Server will run on:
http://localhost:3000
